	Программный модуль APCON (Analysis, Parsing, Classification Of News) предназначен для парсинга новостных источников, обработки естественного языка новостных текстов, а также классификации текстов по категориям на основе собранных новостных статей.

	Программный модуль APCON представляет собой GUI-приложение  (приложение, имеющее графический пользовательский интерфейс) написанное на языке Python. Структурно в данном приложении реализованы следующие направления:
	- Парсинг новостных сайтов;
	- Сохранение полученной информации в базу данных (БД) и получение различных «срезов» из этой БД;
	- Natural Language Processing (NLP) -  обработка естественного языка, как стороннего текста, так и полученных новостных статей;
	- Классификация стороннего текста по категориям собранных новостных статей;

Для корректной работы функции «Парсинг» необходимо:
	- наличие браузера Google Chrome;
	- наличие  в директории с программой приложения chromedriver.exe, версия которого должна СОВПАДАТЬ с версией браузера Google Chrome

	Скачать chromedriver.exe можно с официального сайта:
		- https://chromedriver.chromium.org/downloads
	Перед запуском функции «Парсинг» проверьте выполнение этих условий!
